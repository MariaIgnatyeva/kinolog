# Синтия — методичка (one-pager)

Краткая шпаргалка по итогам встреч 1–6 с кинологом. Подробное описание упражнений — см. **[Упражнения](exercises.md)**.

---

## Базовые принципы

- **Единые правила для всех в семье.** Синтия — «спорящая» собака, она ищет, у кого правила не работают. Если правило есть — соблюдаем все вместе.
- **Регулярность важнее интенсивности.** Идеал — ежедневно, минимум 1–2 раза в неделю.
- **Не тянем за поводок.** Вместо рывка — подходим к собаке, оттесняем корпусом, перекрываем взгляд ладонью.
- **Не стоим, когда говорим «пойдём».** Сами идём первыми или отходим назад/вперёд по поводку. Синти ждёт, что человек двинется.
- **Больше разговариваем с собакой.** Проговариваем, куда идём («налево», «направо», «прямо», «пойдём домой»). Это работает.
- **Голос строже, если обычный игнорирует.** Не громче — именно интонация серьёзнее.
- **Не подкрепляем нежелательное.** Не хвалим, если она сама ушла от мусорки — это уже самоподкрепление.

## Амуниция

- Шлейка, поводок 1.5–2 м.
- **Поводок на переднем кольце** шлейки, если тянет. Цель — постепенно перейти на спинное кольцо.
- На прогулках — намордник (для безопасности с подбором/мелкими).
- Кусочки/корм с собой каждую прогулку.
- На незнакомые/рискованные маршруты — трекер (AirTag).

---

## Выход из дома

1. Сначала одеваем Синти, потом одеваемся сами.
2. Если убегает от шлейки — спокойно зовём в коридор, садим, одеваем за кусочек.
3. Если тянет в подъезде — кружочки на этаже до успокоения (на нескольких этажах, если надо).
4. Из подъезда — выходим **шагом**. Если вылетает — прикрываем дверь, возвращаемся, кружочки, повторяем спокойнее.

## Прогулка и тяга поводка

**Схема длины поводка:**
> супер короткий → шагает → даём 1–1.5 м → шагает → даём ещё больше → если тянет, возвращаемся на супер короткий.

- Иногда сразу даём весь поводок, если она успокоилась и рядом никого нет.
- **Перевозбуждение** → паттерн «раз-два» / «квадратик кусочков» / кружочки.
- **Упирается** → крепко держим поводок и ждём; или подходим вплотную, обходим спереди и оттесняем корпусом в нужную сторону, говорим «пойдём» + похвала.
- **Узкие тропинки = триггер.** Выбираем широкие маршруты (например, к метро). На узких — оттесняем в сугроб на коротком поводке.
- **Метки/занюхивание** → ограничиваем коротким поводком, уводим. Дозываем за собой, если занюхивается.
- **На перекрёстках** → проговариваем направление: «налево», «направо», «прямо».

## Триггеры

| Ситуация | Что делать |
|----------|-----------|
| **Другая собака впереди** | Не зависаем, ускоряем шаг, чуть подтягиваем. На рабочей дистанции отзываем, отбрасываем кусочек вбок (так, чтоб видела), на «можно» подходим |
| **Замерла на собаке** | Обходим спереди, оттесняем корпусом, придерживаем за шлейку |
| **Играет, надо уйти** | Ловим паузу в игре → активно зовём → на короткий поводок → уходим. Можно чуть подтаскивать |
| **Кошки / мелкие** | На рабочей дистанции отзываем в сторону, даём кусочек или игрушку. Берём больший радиус обхода, не прячемся |
| **Орёт на триггер** | Стоим, ждём паузу, в неё зовём дальше |
| **Птицы** | Принцип Премака: отходим на весь поводок, зовём к себе → «можно» → пускаем чуть погонять → возвращаем |
| **Мусорка / подбор** | Стоим, ждём отказа (можем цокнуть/назвать по имени), не зовём активно. Отказалась — кусочек, пошли. Долго не стоим |
| **Копает** | Подходим спереди, оттесняем корпусом назад, придерживаем за шлейку (без оттягивания) |
| **Палки** | Не зависаем, быстро уводим |
| **Люди (хочет обниматься)** | Идём между ними и Синти. Если прыгает — выставляем ногу. Просим людей не провоцировать |
| **Чужой мячик на площадке** | Уходим в самом начале, как только мяч достали. На дистанции можно постоять, потом увести |

## Если убежала

- **Не бегать за ней.** Это игра — мы её подкрепляем.
- Говорим «пока-пока» и **уходим в противоположную сторону**.
- Не звать активно, не кидать кусочки в догонку — всё это подкрепление.
- **Без поводка пока не отпускаем** в знакомых местах с её «закладочными» маршрутами. В незнакомых — она сама держится рядом.
- Трекер на ошейник для подстраховки.

## Площадка / отбор игрушки

- Синти забирает игрушку у других собак, чтобы за ней бегали.
- **Не подходим, не догоняем.** Уходим в обратную сторону, «пока-пока».
- Если ходим на площадку — выходим с неё в самом начале, когда у кого-то достают мяч.

## Игрушки и мячик

- **Косточка** — рабочая, ей пользуемся регулярно.
- **Мячик** — сильно возбуждает, начинаем с малых дистанций, постепенно усложняем.
- **Кролик с пружинкой** — для тренировки самоконтроля дома (имитация мелкой живности).

**Базовая работа с игрушкой:**
- Откидываем вбок → ждём, что посмотрит на нас → «можно» → пускаем.
- Поднимаем игрушку вверх → сидит, не прыгает → отдаём по «можно».
- Машем игрушкой на выдержке «сидеть»/«лежать».
- Откидываем мячик при подзыве, страхуем поводком, дозываем, только потом «можно».

## Команды

| Команда | Что значит |
|---------|------------|
| **«Пойдём»** | Идём дальше. Сами начинаем движение, не стоим |
| **«Ко мне»** | Эталонная команда подзыва. Используем только когда уверены, что подойдёт. Приходит → садится перед нами → ждёт «гуляй» |
| **«Иди сюда» / «подойди, пожалуйста»** | Бытовой подзыв, для сложных ситуаций (где может проигнорить). Не «портим» этим «ко мне» |
| **«Рядом»** | У ноги. Тренируем кусочком от носа, постепенно поднимаем руку выше |
| **«Сидеть» / «ждать»** | Выдержка. Усложняем кусочками, игрушками, отходом в другую комнату |
| **«Можно»** | Разрешение взять/побежать |
| **«Гуляй»** | Свободное поведение / пассивный отказ от еды |
| **«Чисто» / «нет»** | Запрет, строго |
| **«Сугроб»** | По желанию — выученная команда «прыгай в сугроб», для совместной игры |

## Регулярный набор упражнений

Делать в течение прогулки или дома:

1. **Отказ от кусочка с руки**
2. **Отказ от брошенного кусочка (поводок)**
3. **Дорожка с кусочками с остановками** (зигзагом)
4. **Активный + пассивный отказ**
5. **Выдержка с кусочками рядом** (на полу, на лавке, дома)
6. **Выдержка с отходом в другую комнату**
7. **Закладки** (подъезд, лавка, у мусорки)
8. **Вышагивание** со сменой длины поводка
9. **Кружочки / паттерн «раз-два» / квадратик** при перевозбуждении
10. **Игрушка: отброс вбок / подъём вверх / махание на выдержке**
11. **Откат мячика при подзыве**
12. **Принцип Премака с птицами**
13. **Кролик-пружинка на выдержке**
14. **Подзыв «Ко мне»** (эталонный) — в простых условиях
15. **«Рядом»** у ноги
16. **Выдержка + мячик в стороне (два человека)** — Боря катает мяч на дистанции, Маша работает с собакой
17. **Выдержка в магазине** (входная группа с большим тамбуром) — 3–5 минут максимум
